Blog Post #1: Setting up a appealing CSS style to your website in minutes

Hello everyone. I thought I would share how I setup the general style for my website in a couple minutes (ex. Background Color, Font, Font Color, etc...). I think a key to having a smooth looking website is to select a color palette that have colors that flow with each other, the way I get a good color palette quickly is by using colorhunt.co this website has plenty of color palettes to use with your website. After I select a color palette that I like the look of I head over to my CSS file.

Here in my css file I put in a multi line comment the hex colors from the palette

/*
        #303841 : Black
        #3a4750 : Grey
        #00adb5 : Cyan
        #eeeeee : White
*/

I keep this at the top of my css file so I can always refer back to my color palette when setting up my CSS

Now we need a font. I use Google Fonts for this because it has hundreds of free to use fonts

Once you find a font on Google Fonts that you like click on it to expand it then select a style you like, after that hit embed then copy that html code to your html page(s)

Then what you should do is define the core of your websites style (ex background color, font, font color)

so lets start with changing our html element. We are going to want to change the background color to the first and darkest color of our color palette

html{
    background-color: #303841;
}

Then lets set our text color and font. for the text color you are usually going to want to use the third color in your palette

html{
    background-color: #303841;
    color: #eeeeee;
    font-family: 'Sriracha';
}

Pretty much every website has a link somewhere on it, so lets change how are "a" tag are going to look. for this you usually use whatever color contrasts the others in the palette

a{
    color: #00adb5;
    /* text-decoration: none; */
}

Ok not lets setup a head and body class we can use to display text across our webpages. I think the text should be indented a little bit also.

 /* Disclaimer there are better ways to do this I know but this works (:/*
    .head{
        font-family: 'Sriracha';
        font-size: 50px;
        text-align: left;
        padding-left: 250px;
        padding-top: 150px;
    }

    .body{
        font-size: x-large;
        font-family: 'Sriracha';
        text-align: left;
        padding-left: 275px;
    }

Hey wouldn't it look cool if all our elements faded in when the page was loaded? Lets do that.

    /* first setup our fadein method
        @-webkit-keyframes fadein {
            from { opacity: 0; }
            to   { opacity: 1; }

            /* now lets add the fade in to our other elements /*

            .head{
                font-family: 'Sriracha';
                font-size: 50px;
                text-align: left;
                padding-left: 250px;
                padding-top: 150px;
                -webkit-animation: fadein 1s; 
            }

            .body{
                font-size: x-large;
                font-family: 'Sriracha';
                text-align: left;
                padding-left: 275px;
                -webkit-animation: fadein 1.5s; 
            }
        }
    */

Congrats thats the basic CSS of this website! of course there is more than that and if you would like to see it its on My Github but thats the basic CSS and after doing this once or twice you can setup the basic styling of your website in a few minutes.
